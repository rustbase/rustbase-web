# Usage

Rustbase CLI uses the syntax similar to SQL.

-   **Value** - Must be in json format.
-   **Key** - Must be in string format.

## Inserting data

To insert a new record:

```bash
insert <value> in <key>
```

## Updating data

To update a record:

```bash
update <value> in <key>
```

## Delete data

To delete a record:

```bash
delete <key>
```

## Get data

To get a record:

```bash
get <key>
```
