# What is DustData

DustData is key-value storage engine [Rust](https://www.rust-lang.org/) library. Here is some features:

-   **Fast** - DustData is written in Rust. It is fast and safe.
-   **Data Concurrency** - DustData is designed for data concurrency.

# How to install

Add the following to your Cargo.toml:

```toml
[dependencies]
dustdata = "0.2.8"
```

# Usage

## Initialization

```rust
// DustData Configuration
let config = dustdata::DustDataConfig {
    path: "./test_data".to_string(),
    lsm_config: dustdata::LsmConfig {
        flush_threshold: dustdata::Size::Megabytes(128),
    }
};

let mut dustdata = dustdata::initialize(config);
```

## Insert data

```rust
// ...
// Creating a data
let data = bson::doc! {
    "name": "John Doe",
    "age": 30,
}

dustdata.insert("key", data);
```

## Getting data

```rust
// ...
let value = dustdata.get("key").unwrap().unwrap();
println!("{:?}", value);
```

## Updating data

```rust
// ...
let data = bson::doc! {
    "name": "Joe Mamma",
    "age": 42,
}

dustdata.update("key", data);
```

## Deleting data

```rust
// ...
dustdata.delete("key");
```
