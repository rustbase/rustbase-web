# Overview

Which record on Rustbase must be in a database.

# Databases

In Rustbase CLI, you can access to databases by using `!database` command.

```
!database <database_name>
```

## Create a database

Rustbase creates a database automatically when you insert a record to a database.

On a Rustbase CLI:

```
!database <database_name>
insert {"x": 1} in key
```
