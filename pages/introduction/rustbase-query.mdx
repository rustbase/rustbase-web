# Rustbase Query

Rustbase Query (or RBQL) is a language to query Rustbase.

You can use RBQL to query data on Rustbase, making a CRUD Operations.

## Syntax

### CRUD Operations

#### Create

You can create/insert a new data on Rustbase using `insert` keyword.

```
insert <value> in <key>
```

Example:

```
insert "Hello World" in Hello
```

#### Read

```
get <key>
```

Example:

```
get Hello
```

#### Update

```
update <value> in <key>
```

Example:

```
update "Hello Rustbase" in Hello
```

#### Delete

```
delete <key>
```
